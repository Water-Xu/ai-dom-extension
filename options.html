💡 使用提示
  • 点击扩展图标，在对话框输入自然语言描述你想修改的内容
  • 点击 🎯 选择区域 后在页面上点击任意元素，让 AI 聚焦在该元素上
  • 点击 ↩ 撤销 可以恢复上一次 AI 修改前的页面内容
  • API Key 仅保存在本地浏览器存储中，不会上传到任何服务器
  • 支持所有兼容 OpenAI Chat API 格式的服务商
